Project Ideas

Overview

The project I will be building will be a 2D side-scrolling platform game using the Unity program. The user will move the character through some levels to fight a final boss. This will require programming some Javascript or preferably C# to accomplish this. After viewing some basic tutorials and videos I am confident I can complete the project and will learn quite a lot of valuable skills in programming languages and programming tools along the way. This project will provide a great quantity of experience necessary to achieve a job in the programming industry and specifically a job like the one listed above.

Motivation

For years I have enjoyed playing computer, console and mobile games of all types and genres. However, the older I get I find myself becoming less excited for the newer games and more detached from the new gaming generations releases and feel like I could make something more appealing to not only myself but to others of my age group. While this goal is quite a journey away from where my skillset currently lies at the moment, I strongly feel that given the time to learn how to program that my objective can definitely be achieved.

Description

The project I plan to develop will be a 2D side scrolling platform game. There have been many games like this to draw inspiration from however the main reason for choosing to create this type of game in Unity is not only is it a smaller game to build thus lowering the required build and completion time, but there are numerous helpful guides and tutorials and videos online. This project is merely to showcase the skills I have learned so far in this course and will highlight the 13 weeks of study I have done so far. Trying to create a AAA game would require years and years of work so it is important to be realistic with the limited amount of time that I have to complete this project and choose something much, much smaller in size. I personally feel that with the limited time available I can create a small number of levels with a boss battle level at the end.

The game will have a character or hero which will be required to move from left to right across a small number of levels avoiding, jumping and manoeuvring numerous obstacles and occasionally fighting some monsters or some other form of enemy. If the character moves within a specific range of the enemies they will move towards the character and start to attack the character. Upon being hit the character will die and respawn. If they die too many times then it will be game over and they will have to begin the game again. The final level will be a boss battle that upon completion will end the game. The game will have some 8 bit music that is symbolic of the gaming genre at the time and have a small number of other sounds such as jumping noises and enemy noises. The story line will be very basic if existent at all as the goal of this project is not intended to develop my story telling but intended to build on and test my ability to program and create clean and efficient code in a new programming language.

Tools & Technologies

The main programming tool I will be using will be Unity game development platform. Unity is a content creation engine that is able to be used from beginners all the way up to corporate giant companies. It can be used to make both 2D and 3D games and is the worlds most widely used content creation engine. While unity is programmed using C#, users can use Javascript, C# or Boo programming languages to write the code. Unity also provides a number of services such as analytics, certifications, cloud building, collaborations, diagnostics, multiplayer and the asset store.

Skills Required

One of the biggest required skills is obviously programming in either C# or Javascript. While programming in Unity with Javascript is possible the C# language is much more preferable. With this in mind, viewing some basic C# tutorials on websites or YouTube will be a much-needed requirement if I am to complete this project in the easiest and most efficient way possible. Another required skill is the use and control of the unity asset store. While the majority of the asset store costs money to download and use there are some free samples and objects that will greatly improve project efficiency. This will save a lot of time that would otherwise be spent designing or drawing visuals such as playable characters, background images and other utilities and technologies.
